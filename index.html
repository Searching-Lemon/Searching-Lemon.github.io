# 脑袋昏沉

2021-03-25

近来大都难眠多梦，醒时又竟茫然不知所措，头脑昏昏然仿佛坠入沼泽地，又像迈入死胡同。大脑就像一个扁平般的同心圆，仅有小圆能够勉强思索一二，却也是濒临虚弱的姿态，至于小圆之外，头痛如冻土般横贯压迫内圆，所有的浪漫旖旎和情感变化统统如波涛撞碎于山崖脚下，迸碎为无可名状的烦闷和郁积。

偶有激情高涨，仿佛如山峦崩灭、如宙宇演化、又像中二的天使降临和笑颜如花，宣泄之后或泪流、或无趣、或厌世、或满足。抚慰的斯德哥尔摩症时常让我的心绪对中二念念不忘，间或反扑就是几个时辰的翻来覆去。敏感的内心、脆弱的神经仿佛发烧一般不能自己。沉溺中醒来，大都泪流满面，着实不知如何相处。![]

约莫时下午两三点光景，刚醒的我头脑很是恍惚，不消说脑袋痛的要死，大脑就像被封锁了，没有我想想念念的醒来放松舒服的那种感觉。

到下午了也是这样，不知道我的内心紧张感还是没有褪去的缘故，脑子老是不正经的想其他事，其实身体还是有点不舒服，刚感冒的鼻子发酸，老想流眼泪，却又被那种封锁的感觉所禁锢着，怎么着都不是舒坦。

我在想什么呢，我也不清楚。中午怎么睡着的我都不知道，是那种只看自己的想法，还是自己只是平常人的小心思没有什么特殊的正式感觉让我极快入眠，应该还有躺床上立刻随之而来的困意来一并问候。

我还有什么感觉不好的么？睡梦中我又做了什么梦，我竟然没有了半点感觉，现在也回忆不起任何相关零碎的记忆，我在梦中干了什么，潜意识里我有什么欲望，或者是梦魇、恐惧？稀奇古怪的想法和着不能自己的环境，其实醒着我也大多管不住自己hhh，不知道会发酵成什么东西。

专情还是不顾一切的感情奔放啊，谁晓得。我有什么执念、有什么欲望、有什么性幻想、以至于情绪的猛然宣泄到不能自己，一次又一次的掉泪啊。常说“我们总是容易用一种自虐的方式制造出一种痴情的假象来使自己站在感情的道德制高点上，来获得一种畸形的满足感与安全感”这话对我在入眠前的胡思乱想没有更好的描绘了。梦中我踏破千山万水，穷尽铁马冰河，只落得乞儿般的境地。所幸你来找我，我在破落的街道和你破镜重圆...etc.都是这样的过程，感动了我自己，惊艳了我那枯竭的内心。

又或者有美人琴瑟和鸣，遍赏人间美景，等等，不一而足。不知道我在清醒时受了什么糟糕的冷遇，又被过往的哪件事所攫取，最后落得自怜自艾的下场。只能找寻这些幻想的鸦片来维持得过且过的生活。

全身紧绷啊我，不知道该何去何从。时常的我被恐惧贩卖，又被他人指点，亦或者被过去那经历所裹挟，被自己的状态所害。找点乐子，要来白粉么，烟草都不碰的小家伙还是趁早绝了这个念头免得以后自寻烦恼吧。

幻想的我精神焕发，现在的是在其期望某一天上帝大发慈悲睁眼么，凭空而来怕是要砸死我等，不若离去。

随风自兹去。
