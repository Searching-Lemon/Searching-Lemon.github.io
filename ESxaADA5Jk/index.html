# 做梦

2021-03-30

梦到了什么呢，梦中的我满身大汗。

好像是我骑着电动车带着小妹去找一个女孩子，跑了好久，弯弯绕了好远还不敢接近她的跑了过去，场景大概类似一个小镇，最后到了一条小河。我从她的正南绕到了东面隔几户人家过了一条小河，那座桥是短短的短石桥，桥身很低，上面都是土路，在刚下过雨的天气里满是泥泞。我从那里绕到了她的北面，打响笛叫她又被她叫了过去。

镜头一转，画面来到了她家，不知什么时候，我脚下的电动车变成了自行车，我好像很恐惧，在她家门口路过多跑了十几米被她还是喊了回来。见到了她爸，我叫叔叔好像是为了建档立卡户的问题，他用这些问题来问我，最后在我给点关于声音呃或者说关于身子骨的问题的时候生气拂袖离去。期间他的声音一直是刚强的。

老头子有秃顶，可头发确是黑的，穿着黑马褂，胡子留成了银灰色的络腮胡，老人纹不甚多，没看见老人斑，也没有驼背的迹象。不知道为什么，很不喜欢。

走了之后的就是自己的想法了。我埋怨她，好像是要不是这是她爹，换做其他人，我早就不理了。

可是从清醒的现在说，我在恐惧着什么。恐惧于不敢直接面对她，不敢直接去找她，或者不敢去面见她的家人。小妹呢，小妹也丢了。我弄丢了啊。
