# 做梦

2021-03-30

梦到了什么呢，梦中的我满身大汗。

好像是我骑着电动车带着小妹去找一个女孩子，跑了好久，弯弯绕了好远还不敢接近她的跑了过去，场景大概类似一个小镇，最后到了一条小河。我从她的正南绕到了东面隔几户人家过了一条小河，那座桥是短短的短石桥，桥身很低，上面都是土路，在刚下过雨的天气里满是泥泞。我从那里绕到了她的北面，打响笛叫她又被她叫了过去。

镜头一转，画面来到了她家，不知什么时候，我脚下的电动车变成了自行车，我好像很恐惧，在她家门口路过多跑了十几米被她还是喊了回来。见到了她爸，我叫叔叔好像是为了建档立卡户的问题，他用这些问题来问我，最后在我给点关于声音呃或者说关于身子骨的问题的时候生气拂袖离去。期间他的声音一直是刚强的。

老头子有秃顶，可头发确是黑的，穿着黑马褂，胡子留成了银灰色的络腮胡，老人纹不甚多，没看见老人斑，也没有驼背的迹象。不知道为什么，很不喜欢。

走了之后的就是自己的想法了。我埋怨她，好像是要不是这是她爹，换做其他人，我早就不理了。

可是从清醒的现在说，我在恐惧着什么。恐惧于不敢直接面对她，不敢直接去找她，或者不敢去面见她的家人。小妹呢，小妹也丢了。我弄丢了啊。

# 脑袋昏沉

2021-03-25

近来大都难眠多梦，醒时又竟茫然不知所措，头脑昏昏然仿佛坠入沼泽地，又像迈入死胡同。大脑就像一个扁平般的同心圆，仅有小圆能够勉强思索一二，却也是濒临虚弱的姿态，至于小圆之外，头痛如冻土般横贯压迫内圆，所有的浪漫旖旎和情感变化统统如波涛撞碎于山崖脚下，迸碎为无可名状的烦闷和郁积。

偶有激情高涨，仿佛如山峦崩灭、如宙宇演化、又像中二的天使降临和笑颜如花，宣泄之后或泪流、或无趣、或厌世、或满足。抚慰的斯德哥尔摩症时常让我的心绪对中二念念不忘，间或反扑就是几个时辰的翻来覆去。敏感的内心、脆弱的神经仿佛发烧一般不能自己。沉溺中醒来，大都泪流满面，着实不知如何相处。![]

约莫时下午两三点光景，刚醒的我头脑很是恍惚，不消说脑袋痛的要死，大脑就像被封锁了，没有我想想念念的醒来放松舒服的那种感觉。

到下午了也是这样，不知道我的内心紧张感还是没有褪去的缘故，脑子老是不正经的想其他事，其实身体还是有点不舒服，刚感冒的鼻子发酸，老想流眼泪，却又被那种封锁的感觉所禁锢着，怎么着都不是舒坦。
